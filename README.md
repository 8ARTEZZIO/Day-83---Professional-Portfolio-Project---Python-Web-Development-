### How to Run ###

1. Clone the repo:
   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   cd your-repo-name
   ```
2. Install Flask:
   ```bash
   pip install flask
   ```
3. Run the app:
   ```bash
   python app.py
4. Open your browser and go to:
   ```cpp
   http://127.0.0.1:8000/
   ```
